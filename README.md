# 🚀 Interview-AI

Interview-AI is an AI-powered web platform that helps users prepare for job interviews in a smarter and more structured way.  
The platform uses AI to analyze job roles and generate personalized interview questions, ATS-friendly resumes, and study plans.

🔗 **Live Demo:** https://interview-ai-3t9d.onrender.com

---

# ✨ Features

- 🤖 **AI Generated Interview Questions**
  - Generate role-specific interview questions based on the job role.

- 📄 **ATS Friendly Resume Generator**
  - Create professional resumes optimized for Applicant Tracking Systems.
  - Download resumes instantly.

- 📊 **Skill Improvement Suggestions**
  - AI suggests the important skills required for the selected job role.

- 📅 **Day-wise Study Plan**
  - Get a structured preparation roadmap for interview preparation.

- 🔐 **Authentication System**
  - Secure login and logout for users.

---

# 📸 Project Workflow

1. User logs into the platform
2. Enters job role or job description
3. AI analyzes the role
4. Platform generates:
   - Interview questions
   - Skill suggestions
   - Study plan
5. User can generate and download an ATS-friendly resume
